# Worklog template

Copy to `.do-it/worklog/YYYY-MM-DD.md` or `.do-it/worklog/<goal-slug>.md`.

```markdown
# <goal one-liner>

**Date:** YYYY-MM-DD  
**Tier:** Light | Standard | Heavy  
**Status:** active | blocked | done

## Intent

What we are trying to accomplish this session (1–3 sentences).

## Current truth

- Branch / worktree:
- Files touched:
- Commands run:

## Decisions

- Grill/brainstorm pointers (if any):
- User choices locked:

## Next

- [ ] Next concrete step
- [ ] Verification command

## Residual risk

What is not proven yet.
```

Keep handbook stable; put churn here.
